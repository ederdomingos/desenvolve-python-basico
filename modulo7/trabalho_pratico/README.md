# Trabalho Prático – Sistema de Gestão

## Introdução
Este trabalho modela uma empresa de vendas de produtos de informática.
O sistema possui usuários com níveis de permissão (gerente e outros usuários).
Os produtos possuem nome, preço e quantidade.

## Implementação

### Usuários
- Estrutura de dados: lista de dicionários
- Arquivo: usuarios.csv
- CRUD:
  - Create: criar usuário
  - Read: listar usuários
  - Update: não implementado
  - Delete: remover usuário

### Produtos
- Estrutura de dados: lista de dicionários
- Arquivo: produtos.csv
- CRUD:
  - Create: criar produto
  - Read: listar produtos
  - Update: não implementado
  - Delete: não implementado
  - Busca por nome
  - Ordenação por nome
  - Ordenação por preço

## Conclusão
O projeto permitiu aplicar conceitos fundamentais de Python como estruturas
de dados, funções, controle de fluxo e manipulação de arquivos.
